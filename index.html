BAccessibility

Help to make your joomla site more accessibile

The module installation is same as any other Joomla module. You can control the view with some configuration attributes in Joomla management system.

The module should be refer to a specifc location that can be added in the template's index.php file The module will "float" on left or right side of the screen.
